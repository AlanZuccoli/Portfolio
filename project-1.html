Overview do Projeto

Este estudo de caso foi proposto pela Rocketseat, com Licença MIT. A ideia deste estudo de caso foi aprimorar habilidades e obter mais prática. A concepção do projeto foi disponibilizada no figma, com prazo de 4 dias para entrega. Foi concluído em apenas 3 horas de codificação.

Este é o primeiro projeto com design responsivo que criei. Este é um projeto de código aberto e está disponível no GitHub para consulta e feedback.
